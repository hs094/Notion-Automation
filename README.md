# Notion-Automation

[![Youtube DB Sync Job](https://github.com/hs094/Notion-Automation/actions/workflows/sync.yml/badge.svg)](https://github.com/hs094/Notion-Automation/actions/workflows/sync.yml)

Automations that keep Notion databases in sync with external sources. The **YouTube DB sync** runs on a schedule via GitHub Actions, pulling video titles, covers, and channel metadata from yt-dlp into Notion. See [`youtube_db/`](./youtube_db/README.md) for details.
